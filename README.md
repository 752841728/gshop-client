## 项目功能 ## 

> 此项目是移动端外卖项目，拥有用户注册、登录，添加购物车、提交订单、查看订单，查看商家、搜索商家等基本功能，使用vue-cli3脚手架创建的前后端分离项目

## 项目安装 ##

> 开发环境的运行：
> npm install
> npm run dev
> 
> 生产环境的运行：
> 将nginx->html中的文件替换为dist中的文件
